6 ## Overview
7 DepthProjectionMapping is able to project depth image using intel RealSense D4XX series and a projector.
8 Also, project video by specifying the projection area.

31 ## Usage
32 Build this project since Visual Studio 2017 and run.
33 1. Calibrate camera.
34  Note: Once the camera has been calibrated, it will not run from next time on.
35 2. Adjust the camera position so that the marker projected by the projector is within the angle of view of it, and press 'q' or 'd'.
36 d: Start projection mapping of depth image
37 q: Start projection mapping of video by specifying the projection area
38 
39 ### Projection mapping of depth image(Press 'd')
40 1. If the projected depth image is noisy, please modify setting.json in the project according to the environment.
41  "param-depthclampmax": "3400", // Maximum distance to get depth(depthunits * depthclampmax)
42  "param-depthclampmin": "0", // Minimum distance to get depth(depthunits * depthclampmin)
43  "param-depthunits": "1000", // the depth unit value, in micrometre
44 
45 ### Projection mapping of video by specifying the projection area(Press 'q')
46 1. Specify the background image. Click the Set button of "Background image".
47 2. Place the object to be projected in the projection area. The area where there is no difference from the background image will be black.
48  The image will not be projected on the area(mask area).
49  <img src="https://raw.github.com/wiki/tes2840/DepthProjectionMapping/images/DepthProjectionMapping_Background.gif" width="50%">
50 3. Adjust the mask area. The mask area can be adjusted by sliding the Binarization threshold.
51  <img src="https://raw.github.com/wiki/tes2840/DepthProjectionMapping/images/DepthProjectionMapping_Binarization.gif" width="50%">
52 4. Press the Set button of "Mask" when only the object you want to project is displayed. The mask area can be fixed.
53  <img src="https://raw.github.com/wiki/tes2840/DepthProjectionMapping/images/DepthProjectionMapping_FixMask.gif" width="50%">
54 5. Modify the mask area. You can right-click on the screen to exclude the point from the mask area, or left-click to specify it as the mask area.
55  The brush size can be adjusted by adjusting the Thick size slider.
56  <img src="https://raw.github.com/wiki/tes2840/DepthProjectionMapping/images/DepthProjectionMapping_ModifyMask.gif" width="50%">
57 6. In the "screen" window, specify the image you want to project. Right-click to specify the start point, and release at the end point.
58  <img src="https://raw.github.com/wiki/tes2840/DepthProjectionMapping/images/DepthProjectionMapping_CaptureArea.gif" width="20%">
59 7. When all the steps 1~6 are set, press the ""Start" button of "Composition".
60  <img src="https://raw.github.com/wiki/tes2840/DepthProjectionMapping/images/DepthProjectionMapping_GUI.png" width="20%">
61 8. Projection mapping will begin.
62  <img src="https://raw.github.com/wiki/tes2840/DepthProjectionMapping/images/DepthProjectionMapping_Demo.gif" width="20%">
